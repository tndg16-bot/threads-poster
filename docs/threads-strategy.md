### 1.2 ターゲットオーディエンス

**メインターゲット**: 25〜40歳のビジネスパーソン
- AIに興味はあるが、何から始めればいいかわからない
- 自己成長意欲は高いが、情報過多で動けていない
- 転職・副業・独立をぼんやり考えている
- 「意識高い系」に見られたくないが、本音では変わりたい

**サブターゲット**: コーチ/コンサルタント/個人事業主
- 発信力を高めたい
- AIを仕事に組み込みたい
- クライアント獲得チャネルを増やしたい

### 1.4 ポジショニング：差別化戦略

**市場分析**:

| ポジション | 飽和度 | 例 |

| **AI×自己成長の実践記録** | **低** | **ここを取る** |

**差別化の3軸**:

3. **営業マン出身の泥臭さ**: ハイスペック感ではなく「普通の人がステップアップした」ストーリー

**ポジショニングステートメント**:

### 2.4 CTA（Call To Action）ルール

**エンゲージメントベイト回避ガイドライン**:

Threads（Meta）はエンゲージメントベイトを明確にペナルティ対象としている。以下のルールを厳守する。

**NG例（絶対に使わない）**:
- 「いいねしたら〇〇」「保存したら〇〇」
- 「フォローしてね！」の直接的な依頼
- 「リポストお願いします」
- 「コメントで〇〇って書いて」
- 絵文字だけのコメントを促す行為

**OK例（自然なCTA）**:
- 「みなさんはどう思いますか？」（本文の流れで自然に問いかける）
- 「同じ経験した人いるかな」（共感の延長線上）
- 「こういう工夫してる人いたら教えてほしい」（情報交換の姿勢）
- セルフリプライで「詳しくはブログに書きました→URL」

**セルフリプライ戦略**:
- 本文中にはURLを入れない（Threadsはリンク付き投稿のリーチを下げる傾向）
- 投稿後すぐにセルフリプライで「もっと詳しく知りたい方はこちら→記事URL」
- セルフリプライは敬語すぎず、カジュアルに

| 項目 | 内容 |

| 日付 | 2026-04-07（火） |

| カテゴリ | AI活用 |
| 元記事 | `prompt-practice-guide` |
| トピックタグ | `ChatGPT` |

**投稿文**:

ChatGPTを「なんか微妙」と感じてる人、プロンプトの書き方を変えるだけで別物になる。

仕事で即使えるプロンプトの型を3つだけ共有。

① 役割指定型
「あなたは○○の専門家です。○○について、初心者にもわかるように説明してください」
→ 回答の精度が段違いに上がる

② 制約条件型
「300文字以内で」「箇条書き5つで」「メリットとデメリット両方挙げて」
→ 欲しい形式で返ってくる

③ 壁打ち型
「この企画の弱点を3つ指摘して、改善案もセットで出して」
→ 一人ブレストの相手として最強

ポイントは、AIに「何をしてほしいか」だけじゃなく「どんな立場で、どんな形式で」まで伝えること。

人間への指示と同じで、具体的なほど良い結果が返ってくる。
